Política de Privacidad - Tipificaciones Salesforce

Última actualización: Julio 2025

DIVULGACIÓN DE DATOS DEL USUARIO

Esta extensión accede temporalmente al contenido de los campos de texto ÚNICAMENTE cuando el usuario activa la función con Alt+Q.

Datos que se procesan:

  • Contenido del campo de texto activo (solo cuando se presiona Alt+Q)
  • Teclas presionadas para detectar el atajo Alt+Q

USO DE LOS DATOS

  • Los datos se procesan localmente en el navegador
  • Se usan ÚNICAMENTE para buscar y aplicar tipificaciones
  • NO se almacenan, guardan ni transmiten a ningún servidor
  • Los datos se descartan inmediatamente después de su uso

La extensión NUNCA:

  • Almacena datos del usuario
  • Envía información a servidores externos
  • Recopila datos sin acción explícita del usuario (Alt+Q)

DATOS EXTERNOS

La extensión descarga una lista de tipificaciones desde Google Sheets, pero esta lista NO contiene ningún dato personal del usuario.

DECLARACIÓN DE PERMISOS

Esta extensión utiliza los siguientes permisos:

  • activeTab: Para detectar cuando el usuario presiona Alt+Q
  • scripting: Para inyectar el script en páginas de Salesforce
  • host_permissions: Para funcionar únicamente en dominios de Salesforce

CUMPLIMIENTO

  • NO vendemos ni transferimos datos a terceros
  • NO usamos datos para publicidad
  • NO determinamos solvencia crediticia
  • Los datos se procesan ÚNICAMENTE con el propósito de buscar y aplicar tipificaciones

Contacto: Si tienes preguntas sobre esta política de privacidad, puedes contactarnos a través de GitHub.
